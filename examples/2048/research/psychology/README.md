# 2048: why it feels dangerously addictive

A companion to [../history/](../history/). The historical question is
"where did 2048 come from"; this section's question is "why is it so
hard to put down."

There is real literature here, but a lot of it is pop-science press
from the spring 2014 viral wave, and the press largely converged on a
single neurologist's explanation. So this bundle does two things:
walks through the standard accounts, and then -- because they all
quote the same sources -- adds a chapter on where the evidence
actually thins out.

## The short version

Four mechanisms come up over and over:

1. **Dopamine on prediction and on optimal challenge.** Every move is
   a small prediction. The brain releases dopamine on prediction
   regardless of whether the prediction was correct. The difficulty
   curve keeps you in the comfortable middle where dopamine production
   stays high.
2. **Variable-ratio reward schedule.** Tiles spawn randomly (90%
   chance of a 2, 10% of a 4); good positions can be ruined by an
   unlucky spawn, lucky spawns rescue bad positions. This is the same
   reinforcement schedule that slot machines run on, and it is the
   most behaviorally sticky one known.
3. **Low-friction compulsion loop.** Sessions are short. Restarts are
   free. The "one more game" cost is genuinely tiny, which is exactly
   the property that makes the next restart inevitable.
4. **Near-miss math.** The end-state is technically reachable but
   astronomically improbable. Every game ends in "almost." The brain
   treats near-misses as partial wins, not as losses.

The reasons these stack instead of overlapping is that they operate on
different timescales -- per-move, per-merge, per-session, and across
sessions.
